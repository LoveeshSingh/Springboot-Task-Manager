# Spring Boot Task Manager API

A simple REST API built using Spring Boot for learning CRUD operations, service layer, repository layer, and clean architecture.

## Features
- Create tasks
- Update tasks
- Delete tasks
- Mark tasks as complete
- Get all tasks

## Tech Stack
- Java 17+
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL / H2

## How to Run
1. Clone the repo
2. Open in IntelliJ / VS Code
3. Run the main Spring Boot application
